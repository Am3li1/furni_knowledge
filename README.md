# Furniture Admin Interview System

## Setup Instructions

### Prerequisites:
- Python 3.8 or higher
- Internet connection (to connect to database)

### Installation:
1. Open Command Prompt/Terminal in this folder
2. Install dependencies: `pip install -r requirements.txt`
3. Run the system: `python admin_interview.py`
4. Open browser: http://localhost:5001/admin

### Login:
Use the admin password provided separately.

### Troubleshooting:
If `pip install` fails:
- Try: `python -m pip install -r requirements.txt`
- On Windows, you might need to install Visual C++ Build Tools

If you see database connection errors:
- Check your internet connection
- Contact support if the issue persists
